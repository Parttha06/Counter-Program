## ğŸ’» How It Works

The counter uses JavaScript to:
1. Listen for button clicks (increment/decrement)
2. Update the counter value
3. Display the updated value on the screen

```javascript
// Simple counter logic
let count = 0;

increment.addEventListener('click', () => {
    count++;
    display.textContent = count;
});

decrement.addEventListener('click', () => {
    count--;
    display.textContent = count;
});
```
